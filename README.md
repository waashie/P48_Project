# P48 Project

## Overview

The P48 Project is a responsive website developed as part of the SDEV 280 course. The site includes a home page, team information, a working agreement page, and an interactive schedule builder used for planning advertising campaigns. The project demonstrates web development concepts using HTML, CSS, and JavaScript while emphasizing teamwork and project management.

* `Design2.js` – General site functionality
* `Schedule2.js` – Schedule calculator functionality

## Running the Project

1. Clone the repository.
2. Download all project files into the same folder.
3. Open `index.html` in a web browser.
4. Use the navigation menu to explore the site and access the schedule tool.
